Music Library UI

Explore Your Favorite Music Albums

I recently created a Music Library UI using React and Vite to deepen my understanding of React. Having previously worked with Angular, I found the component-based architecture familiar, but learning how React manages state with hooks like useState has been incredibly insightful.

The application allows users to select albums and view details, such as cover images and track lists. I developed custom components like Albums, Gallery, and Tracks, with useState managing the currently selected album to create a dynamic and responsive UI. Users can also add more music to the library, which updates immediately, demonstrating the benefits of props and useState for state management and data flow between components.

Key Learning Highlights include:

- Using React hooks like useState for effective state management

- Vite's impressive performance for fast builds and a smooth development experience

I'm excited to continue expanding this project and applying the knowledge I've gained to future work.
